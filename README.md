# Go Task Manager (CLI + Web API)
A 3-hour sprint project learning Go basics, JSON persistence, and HTTP servers.

## Features
- **CLI Menu:** Add, show list, mark done, and delete tasks.
- **Persistence:** Tasks are saved to a `tasks.json` file.
- **Web API:** Serves task data as JSON.
- **Live Frontend:** An auto-refreshing HTML dashboard every 2 sec.

## How to Run
1. `go run main.go`
2. Open `http://localhost:8080` to see the live feed.
